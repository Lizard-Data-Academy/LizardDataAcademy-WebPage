SQL & Bases de datos

Aprende a pensar, estructurar y consultar datos con SQL: desde qué es una base de datos hasta resolver preguntas reales de negocio. Todo el curso se trabaja sobre DuckDB, una base de datos analítica ligera pensada para aprender haciendo.

¿De qué va este curso?

De no saber qué es una base de datos a resolver preguntas de negocio con SQL

Este curso está pensado para construir una base sólida: primero entenderás qué son los datos y cómo se organizan, después aprenderás a diseñar y crear estructuras, y por último dominarás SQL para consultar, cruzar y analizar información real.

Trabajarás siempre con DuckDB, un motor de base de datos analítico, ligero y que corre en local sin necesidad de instalar servidores. Es la forma más directa de practicar SQL de verdad desde la primera clase, con datasets que simulan escenarios de negocio reales (ventas, clientes, productos).

¿Por qué DuckDB?

  • No necesitas instalar ni configurar un servidor de base de datos.
  • Sintaxis SQL estándar, muy similar a la de motores como PostgreSQL.
  • Pensado para análisis de datos: rápido incluso con archivos grandes.
  • Lo que aprendas se transfiere directamente a otros motores del mercado.

Proyecto final

Un proyecto completo de principio a fin

Aplicarás todo lo aprendido en un proyecto real, en una de estas dos modalidades:

Opción A

Base de datos proporcionada

Recibes una base de datos completa (por ejemplo, de un e-commerce, una cadena de tiendas o una empresa SaaS). Debes explorarla, comprender su modelo de datos y resolver una serie de retos de negocio mediante SQL.

Opción B

Base de datos construida por ti

Diseñas un pequeño modelo de datos a partir de un escenario propuesto o de un conjunto de datos propio: creas las tablas, defines las relaciones, cargas la información y desarrollas las consultas necesarias.

¿Qué entregas al finalizar?

  • El modelo de datos.
  • El script de creación de tablas.
  • El script de carga de datos.
  • Un conjunto de consultas SQL documentadas.
  • Un breve informe con las conclusiones obtenidas.

Al finalizar la formación

Esto es lo que sabrás hacer

  • Diseñar y crear un modelo de datos relacional desde cero.
  • Escribir consultas SQL desde básicas hasta avanzadas (joins, subconsultas, CTEs, funciones de ventana).
  • Explorar y documentar cualquier base de datos con information_schema.
  • Resolver preguntas de negocio reales: ventas, clientes, productos y KPIs.
  • Trabajar con DuckDB como motor analítico de referencia.
  • Aplicar buenas prácticas de modelado, creación y manipulación de datos.
  • Entender y transferir estos conocimientos a otros motores SQL del mercado.
  • Tener un proyecto completo y documentado para tu portfolio.
